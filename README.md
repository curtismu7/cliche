## Install

**Option 1 ‚Äî download the app** (no developer tools needed). Grab either file from the [latest release](https://github.com/curtismu7/cliche/releases/latest):

- **`Cliche-x.x.x.zip`** ‚Äî unzip and double-click **`Install Clich√©.command`**; installs to **`~/Applications/Cliche.app`** (your personal Applications folder ‚Äî no admin password), clears the Gatekeeper flag, and offers launch-at-login. **Recommended.**
- **`Cliche-x.x.x.dmg`** ‚Äî drag to Applications (system folder). Same app; zip installer is simpler if you don't want `/Applications`.

**Option 2 ‚Äî Homebrew** (installs to `/Applications/Cliche.app` ‚Äî pick **either** Homebrew **or** zip/`make install`, not both):

```sh
brew tap curtismu7/cliche
brew install --cask cliche
```

(Homebrew asks you to `brew trust curtismu7/cliche` once ‚Äî standard for third-party taps.)

**Option 3 ‚Äî build from source** (macOS 14+, Xcode Command Line Tools):

```sh
git clone https://github.com/curtismu7/cliche.git
cd cliche
make install    # builds, installs to ~/Applications/Cliche.app, launches
```

**Where is the app?** Zip and `make install` ‚Üí **`~/Applications/Cliche.app`**. Homebrew ‚Üí `/Applications/Cliche.app`. Only one ‚Äî duplicates break Screen Recording permission.

**Permissions** (macOS asks once each): *Screen Recording* on your first screenshot, and *Accessibility* only if you use direct paste. Everything else works with no permissions at all.

**Screen Recording keeps opening Settings even though Clich√© is toggled ON?** macOS grants permission **per app path and per build signature**. Keep **one** copy at **`~/Applications/Cliche.app`**, run `Scripts/fix-screen-recording.sh`, toggle Clich√© **off ‚Üí on** in System Settings ‚Üí Privacy & Security ‚Üí **Screen & System Audio Recording**, then **quit + reopen twice**.

## What it does

### üìã Clipboard history

- Remembers your last **150 text snippets and 50 images** (both configurable in Settings) from anywhere on your Mac; history survives restarts.
- **Fuzzy search** ‚Äî the panel opens with search focused; `hw` finds "hello world".
- **Keyboard-first** ‚Äî `‚Üë‚Üì` select, `‚Ü©` copies, `‚åò1‚Äì9` grab the first nine, `‚åò‚å´` deletes, `‚å•P` pins, `‚å•U` unpins.
- **Paste directly into the app you were using** ‚Äî `‚å•‚Ü©` or ‚å•-click types the item where your cursor was.
- **Pin** anything to keep it forever ‚Äî pins live in their own section at the top, above a "Recent" separator, with an **Unpin All** button; pinned items never count against history limits. Plus **edit text clips in place** and **preview** long text or images in a floating window with copy/pin/edit corners.
- **Images in a horizontal strip**, previewable, pinnable, annotatable.
- **Snippets** ‚Äî reusable templates with `%DATE%`, `%TIME%`, `%CLIPBOARD%` variables.
- **Privacy built in** ‚Äî anything copied from password managers (concealed/transient pasteboard types) is never recorded, with a user-editable ignore list.
- **‚å•1** ‚Äî floating clipboard list at your cursor (Maccy-style).
- **‚åÉ‚å•‚åòC** ‚Äî same floating list (alternate shortcut; customizable in Settings).
- **Import history from Maccy, Paste, or Clipy** ‚Äî open Settings ‚Üí General and click *Import from ‚Ä¶* to migrate your existing clipboard history. Duplicates are skipped; nothing is changed in the source app. The button only appears for apps whose storage Clich√© detects on your Mac.

### üì∑ Screen capture

- **Region capture on a frozen screen** with a magnifier loupe, live pixel-size label, and Shift-to-square ‚Äî plus window, full-screen, and timed (3/5/10 s) capture.
- **All-in-one capture** ‚Äî one hotkey (`‚åÉ‚å•‚åò3`) opens the overlay with a mode strip: Region, Window, Full Screen, or Copy Text, switchable with keys 1‚Äì4.
- **Hide desktop clutter** ‚Äî a Settings toggle excludes desktop icons from captures and recordings; your wallpaper stays.
- **Multi-window capture** ‚Äî pick several windows and capture just those together, everything else excluded.
- **Combine captures** ‚Äî stitch screenshots side-by-side, stacked, or in a grid from the Captures tab.
- **Repeat last region** with one hotkey ‚Äî perfect for iterating on the same area.
- **OCR** ‚Äî select any region, the text in it lands on your clipboard (on-device Vision).
- **Scrolling capture** ‚Äî select a region, scroll the content, Clich√© stitches one tall image.
- **Screen recording** ‚Äî region to MP4, with optional GIF export.
- **Pixel ruler** ‚Äî hover snaps to UI element edges and shows dimensions; drag measures; click copies.
- **Annotation editor** ‚Äî arrows, lines, boxes, ellipses, freehand, highlighter, text, pixelate, **true blur** (unrecoverable), counter badges, and **one-click auto-redaction** of emails/links/phone numbers/API keys. Saved captures stay **re-editable** ‚Äî reopen one and your layers are still there.
- **Beautify panel** ‚Äî make any capture social-ready with a live preview: editable gradient backdrops (colors + angle), padding, rounded corners, shadow, and an optional matte border; **browser/device frames** ‚Äî Safari-style bar (light/dark) with editable URL, Mac window bar, or phone/tablet bezel; **auto-balance** trims uneven margins; export at exact social sizes (X 1600√ó900, Square, IG Portrait); save your look as a **named preset**.
- **Quick Access Overlay** ‚Äî post-capture thumbnail you can drag into Slack/Mail or click to annotate; QR codes in captures get a "copy link" button.
- **Color picker** with hex copy and a WCAG contrast checker; before/after GIFs from any two captures.
- Screenshots land on the **Desktop + clipboard + Captures tab** (format and clipboard behavior configurable).

## Automation

Drive Clich√© from Raycast, Shortcuts, Alfred, or the terminal via its URL scheme:

| URL | Action |
| --- | --- |
| `cliche://capture` (or `?mode=region`) | Capture a region |
| `cliche://capture?mode=window` | Capture a window |
| `cliche://capture?mode=fullscreen` | Capture the full screen |
| `cliche://capture?mode=allinone` | All-in-one capture overlay |
| `cliche://ocr` | Copy text from screen |
| `cliche://repeat` | Repeat the last region |
| `cliche://panel` | Open the clipboard panel |

Example: `open "cliche://capture?mode=region"`

## Signing

Clich√© is ad-hoc signed ‚Äî there's no Apple Developer certificate behind it, which is why Gatekeeper asks for a right-click ‚Üí Open on first launch. The installer clears the quarantine flag for you. Build from source and there's no prompt at all.
